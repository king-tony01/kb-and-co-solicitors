## Technical Specifications for All Images

### Image Dimensions:

- Hero/Section images: 1200x800px (3:2 ratio)
- Service cards: 800x800px (1:1 ratio)
- Minimum resolution: 300 DPI for print quality

### File Format:

- Export as: JPG (for photos) or PNG (if transparency needed)
- Optimize for web (reduce file size while maintaining quality)

### Color Profile:

- sRGB color space for web display
- Apply color grading with purple (#464158) overlay at 10-20% opacity
- Maintain natural skin tones while adding brand color atmosphere

### Photography Style:

- Natural to slightly dramatic lighting
- Shallow depth of field (f/1.8 to f/2.8) for portraits
- Clean, uncluttered backgrounds
- Professional corporate aesthetic
- Diverse, inclusive representation

### Post-Processing:

- Color grade with purple/slate tones
- Slight vignette for depth
- Clarity and sharpness enhancement
- Maintain professional, trustworthy mood
